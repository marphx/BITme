Guide

This workflow is designed so the output can be placed into GitHub repositories, websites, app interfaces, and game asset pipelines.

1. Capture or Upload

Use the camera for a fresh photo or upload an existing image. Uploaded photos keep their original width and height before conversion.

2. Convert Accurately

Choose Original photo dimensions or a pixel-detail preset, then choose a palette limit. The app samples the real image, builds a color palette, and maps each pixel to the nearest indexed color.

3. Export for Use

Download PNG or SVG for websites, CSS box-shadow art for design systems, JSON for app workflows, or a C header for retro-style game code.
