Quick Guide
This app is designed to produce practical pixel assets, not just a filter effect.

1. Add Image

Start the camera and capture a photo, or upload one directly from your device.

2. Convert

Choose a sprite size and palette depth, then convert the image into indexed pixel data.

3. Export

Download PNG, JSON, SVG, CSS, or C header output depending on where you want to use it.

4. Use It

Place the exported asset into websites, GitHub Pages, design systems, or game code pipelines.

What this app does

It captures or uploads a real image, converts it into indexed pixel art, and exports formats that can be reused as functional design elements and game assets.

What makes it accurate

The app uses square source cropping, palette quantization, nearest-color indexing, and pixel-perfect preview rendering so the output matches the exported code.
